先安装 api server: https://github.com/lincenying/mmf-blog-api

```
// 1. 安装依赖
npm install  or  yarn

// 2.1 生成静态文件
npm run build

// 2.2 开发模式
npm run dev

或者

npm run start
```

首页
http://localhost:8080
