# mmf-blog react版

本版本已不再更新, 请移步新版本: https://github.com/lincenying/mmf-blog-react-v2

---

先安装 api server: https://github.com/lincenying/mmf-blog-api

```
// 1. 安装依赖
npm install  or  yarn

// 2.1 生成静态文件
npm run build

// 2.2 开发模式
npm run dev

或者

npm run start
```

首页
http://localhost:8080

登录
http://localhost:8080/login.html
